Civic Starter

A clean, forkable starter template for the Civic Design System. Everything you need to launch an accessible, mobile-friendly site built on the U.S. Web Design System (USWDS).

Getting Started

  1. Install dependencies 
    npm install
  2. Build all assets 
    npm run build
  3. Start development server 
    npm run serve

How This Works with civic-guide

This starter template is the companion to civic-guide, which is the full reference site for the Civic Design System. The two repos are designed to work together:

  • civic-guide — The reference site. Use it to explore components, customize your theme settings, and preview changes in context.
  • civic-starter (this repo) — Your clean starting point. Fork this, copy your customized styles from civic-guide, and build your site.

Workflow

  1. Customize your theme in civic-guide (colors, typography, spacing, components)
  2. Copy your src/styles/ folder from civic-guide into this project
  3. Run npm run build to compile your customized theme
  4. Start building your pages

What's Included

  • USWDS 3.13.0 — The complete U.S. Web Design System
  • Civic theme — Customizable settings in src/styles/uswds-settings/
  • Build system — Gulp 5 + Dart Sass + PostCSS (autoprefixer + minification)
  • Sourcemaps — For easier debugging in browser dev tools
  • Watch mode — Auto-recompile when you edit styles

Scripts

npm run build          # Clean and build all assets
npm run build:styles   # Compile styles only
npm run watch:styles   # Watch for changes and recompile
npm run serve          # Start development server (port 3001)
npm run clean          # Remove dist/ folder

Required Files

Include these in your HTML pages:

<!-- CSS (in head) -->
<link rel="stylesheet" href="dist/uswds/css/uswds.min.css">
<link rel="stylesheet" href="dist/civic/styles/civic.css">

<!-- JavaScript (before closing body tag) -->
<script src="dist/uswds/js/uswds.min.js"></script>

Project Structure

civic-starter/
├── src/
│   ├── styles/              # SCSS source files
│   │   ├── uswds-settings/  # Theme settings (colors, typography, etc.)
│   │   ├── _bridge.scss     # Connects settings to USWDS
│   │   ├── _packages.scss   # Choose which components to include
│   │   ├── custom-styles.scss # Your custom CSS overrides
│   │   └── index.scss       # Main entry point
│   ├── fonts/               # Custom web fonts (if any)
│   └── images/              # Logo and image assets
├── dist/                    # Compiled output (generated by build)
│   ├── civic/               # Your themed assets
│   │   ├── styles/          # civic.css + civic.min.css + sourcemaps
│   │   ├── fonts/           # Copied fonts
│   │   └── images/          # Copied images
│   └── uswds/               # Stock USWDS assets (css, js, fonts, img)
├── gulpfile.js              # Build configuration
├── package.json             # Dependencies and scripts
└── index.html               # This page
