# uswds-starter

A clean, forkable starter template built on the U.S. Web Design System (USWDS). Vanilla HTML/CSS with everything you need to launch an accessible, mobile-friendly siteâ€”styles, structure, and build system in one repo. No framework required. Perfect for anyone building for the public good.

## Getting Started

```bash
npm install
npm run build
npm run serve
```

## Scripts

- `npm run build` - Clean and build all assets
- `npm run build:styles` - Compile SCSS only
- `npm run serve` - Start local development server
- `npm run clean` - Remove dist and work folders

## Structure

- `src/` - Source files (styles, fonts, images)
- `fpac-website/` - Documentation site templates
- `dist/` - Compiled output (generated)
